量表电子化项目

量表电子化需求诞生于汪建对认知评估量表对电子化需求。具体要求：将MOCA量表等量表写成软件或响应式网页，实现一致性、稳定等志愿者、患者认知功能评估功能。

可能采用的技术框架、语言

  • swift语言
  • javascript语言
  • html语言
  • css
  • sql
  • python语言

具体实现方法

思路一

swift实现原生ios应用，数据库内置

  • swift语言

思路二

前端语言结合后端数据库技术，跨平台应用

  • 主要语言：html css js
  • 框架：react框架

主要功能

  • 志愿者登记
  • 自助性量表填写
  • 自动化量表打分、数据处理
  • 量表原始数据、处理后数据可视化
  • 便捷导出模块

需要的知识

  • 前端开发技能
  • 后端数据库技能
